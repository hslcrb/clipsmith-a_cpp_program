## ğŸš€ Key Features

- **Smart Detection**: Automatically identifies JSON, Base64, URLs, and Emails.
- **Premium UI**: Sleek Dark Mode design with intuitive toolbars.
- **Developer Tools**:
  - **JSON Prettify**: Format messy JSON strings with one click.
  - **Base64 Decode**: Instant decoding of encoded data.
- **History Management**: Persistent storage using SQLite with search functionality.
- **Background Tray**: Runs quietly in your system tray without cluttering your workspace.
