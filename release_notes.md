# Clipsmith v1.0 - The Official Release 🚀💎

리눅스 민트 사용자와 개발자를 위한 스마트 클립보드 매니저, **Clipsmith**의 첫 번째 정식 버전을 출시합니다! 🌊✨

---

## 🌟 주요 기능 (Key Features)

### 💎 프리미엄 UI/UX (High-End Visuals)
- **Glassmorphism & Frutiger Aero**: 향수를 불러일으키는 투명하고 청량한 미학과 현대적인 유리 질감의 완벽한 조화.
- **Ultra Polish**: 눈이 편안한 화이트 텍스트 UI와 부드러운 하이라이트 효과.
- **Custom Scrollbars**: 배경과 어우러지는 투명하고 둥근 프리미엄 스크롤바.

### 🧠 스마트 텍스트 엔진 (Smart Intelligence)
- **자동 감지**: JSON, Base64, URL, 이메일을 즉시 식별합니다.
- **원클릭 가공**: 복사된 데이터를 분석하여 JSON 정리, Base64 디코드 기능을 즉시 제공합니다.
- **SQLite 기반 저장**: 클립보드 히스토리를 안전하게 영구 저장하며, 실시간 검색이 가능합니다.

### 🌍 글로벌 지원 (Global Support)
- **한영 병기 UI**: 모든 메뉴와 안내가 한국어와 영어로 동시에 제공됩니다.
- **다국어 코드**: 소스 코드 전체에 한/영 병기 주석이 포함되어 있습니다.

## 🛠️ 설치 방법 (Installation)
1. 필수 패키지 설치: `sudo apt install qt6-base-dev qt6-base-private-dev libqt6sql6-sqlite`
2. 실행: `./run.sh`
